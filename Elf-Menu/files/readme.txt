Стили класть в папку ElfMenu/Styles в памяти телефона.

Настройки стилей:

x - десятичное число
h - шестнадцатеричное число (с ; на конце)
b - 0 или 1 (выкл. или вкл.)

Цвет пишется в формате 0xAARRGGBB;

Выбор типа анимации выделенной иконки:

Непосредственно тип
[selected_type] x

x:
0 - просто сдвиг иконки по Y
1 - "прыгание" иконки по Y
2 - просто увеличение иконки
3 - циклическое увеличение иконки
4 - "прыгание" иконки по X
5 - просто сдвиг иконки по X

Скорость движения иконки по Y
[icon_y_speed] x
Минимальный отступ иконки по Y
[icon_max_y_offset] x
Максимальный отступ иконки по Y
[icon_min_y_offset] x


Скорость движения иконки по X
[icon_x_speed] x
Минимальный отступ иконки по X
[icon_max_x_offset] x
Максимальный отступ иконки по X
[icon_min_x_offset] x

Минимальное и стандартное увеличение иконки
[min_scale] x
Максимальное увеличение иконки
[max_scale] x
Скорость увеличения
[scale_speed] x

Цвет фона:
[back_color] x

Отступ всего меню от верхнего края:
[y_offset] x

Горизонтальный отступ между иконками:
[x_spacing] x

Вертикальный отступ между иконками:
[y_spacing] x

Кол-во столбцов:
[x_tab] x

Ширина иконки:
[x_size] x

Высота иконки:
[y_size] x

Ширина выделенной иконки:
[x_size_s] x

Высота выделенной иконки:
[y_size_s] x

Скорость курсора:
[cursor_speed] x

Цвет курсора:
[cursor_color] h

Ширина курсора:
[cursor_w] x

Высота курсора:
[cursor_h] x

Стиль надписей:
[lable_style] x

где x = 0 - надпись рисуется в заданных координатах
    x = 1 - надпись рисуется под каждой иконкой
    x = 2 - надпись рисуется и в заданных координатах и под каждой иконкой

Шрифт надписи:
[lable_font] h

Цвет надписи:
[lable_color] h

X-координата надписи:
[lable_x] x

Y-координата надписи:
[lable_y] x

Скорость выезжания меню:
[appear_speed] x

Смещение по y четных столбцов иконок:
[eventab_offset] x

Скорость прокрутки:
[scroll_speed] x



Опции:

Есть ли выделенная иконка:
[is_selected] b

Рисовать курсор картинкой:
[is_cursor_img] b
